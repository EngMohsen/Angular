Watcher

Admin interface for configuration and management The purpose of this document is to describe the requirement for watcher the management interface. Watcher is a generic UI management tool that can work on distributed system to manage and distribute configuration as well as monitor and manage operations.
Watcher will depend on the following configuration details.

  • Application management rest interface
  • Application deployment server
  • Application path
  • Server credentials
  • Server credentials
  • Configuration type (yaml / properties)
